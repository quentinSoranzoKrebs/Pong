Bienvenue dans Pong !

1. Installation :
   - Ce jeu ne peut fonctionner que sous windows.
   - Clonez ce dépôt avec la commande : git clone [URL du dépôt]
   - Accédez au répertoire du téléchargement
   - Executez le Pong.exe

2. Utilisation :
   - Exécutez le programme en exécutant Pong.exe.
   - A chaque execution ce programe instalera les mises a jour.

3. Problèmes connus :
   - Ne peut pas s'executer en administrateur.

4. Contact :

Merci d'utiliser Pong ! J'espère que cela répond à vos besoins.
